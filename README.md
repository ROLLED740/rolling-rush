## Features

- 🌍 **8 living worlds** — Jungle, Mountains, Snow, Ice, Waterworld, Junkyard, Pyramids, Landmarks — cycling forever
- ⚽ **33 ball skins** — gradients, soccer, basketball, the full pool set, plus robot, ninja, alien, pirate & skull characters
- 🔥 **Flame Ball** torches boulders for bonus coins · 😇 **Angel Ball** glides over holes
- 🚀 **Boosts** — Head Start, Shield, Coin Doubler, Slow-Mo
- 🏆 **Global leaderboard** with accounts & cross-device cloud saves
- 📱 Works on phone & desktop — touch and keyboard

## Controls

| Action | Touch | Keyboard |
|---|---|---|
| Steer | drag left/right | ◀ ▶ or A / D |
| Speed up / brake | drag up / down | ▲ ▼ or W / S |
| Pause | ⏸ button | Esc |
